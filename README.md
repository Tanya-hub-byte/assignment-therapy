# Dr. Maya Reynolds - Santa Monica Therapy Redesign

A high-fidelity, responsive Next.js 14 website designed for a licensed clinical psychologist. This project focuses on a "Steady Mind" aesthetic, utilizing a Santa Monica-inspired color palette and editorial layouts.

## ‚ú® Features
- **Santa Monica Theme**: Custom color palette (Deep Indigo #3E3A5D, Soft Ivory #F7F2EE) and Gopher/Serif typography.
- **Task-2 Navigation**: Dynamic scroll-aware header that disappears 80ms after scroll and reappears after 1 second for a clean reading experience.
- **Pixel-Perfect Quad-Grid**: A custom-engineered office section where images touch perfectly at central corners, matching the requested landscape dimensions.
- **Arch Bio Section**: Custom CSS Arch shape for the primary portrait with a clean, overlapping floral accent.
- **Global Scroll Transitions**: Implemented a reusable `Reveal` component using Intersection Observer for consistent fade-in animations.
